
DigitalCalm SL is a privacy-first mobile app that helps Sri Lankan smartphone users manage screen time, reduce digital fatigue, and stay focused.

It uses autonomous AI-driven behavior analysis to understand user habits and take smart actions to improve digital well-being — while keeping all sensitive personal data stored locally on the device.

## 🚀 Vision

To build Sri Lanka’s first privacy-focused digital well-being companion that empowers users to take control of their screen habits without sacrificing personal data.
