# DigitalCalm SL

DigitalCalm SL is a **privacy-focused mobile application** designed to help smartphone users monitor screen time, reduce digital fatigue, and build healthier digital habits.

The app analyzes user device usage patterns and generates **personalized mindfulness nudges using AI** to encourage balanced technology use. Usage behavior is processed through a backend pipeline built with **Firebase Cloud Functions**, **LangChain** for AI workflow orchestration, a **Neo4j graph database** for behavioral pattern detection, and a **Large Language Model (LLM)** to generate supportive digital well-being messages.

---

# Key Features

- ✅ View **daily and weekly screen time**
- ✅ Track **most-used apps**
- ✅ Set **app usage limits**
- ✅ Start and manage **focus sessions**
- ✅ Receive **AI-generated personalized digital wellness messages**
- ✅ Detects **high device usage** and sends notifications when usage exceeds healthy limits
- ✅ Identifies **overused apps** and encourages healthy breaks through proactive notifications

---

# How It Works

DigitalCalm SL analyzes device usage behavior and processes insights through an AI-powered backend pipeline:

1. User device usage data is collected (privacy-aware).
2. Usage patterns are processed using **Firebase Cloud Functions**.
3. Behavioral relationships are analyzed using **Neo4j Graph Database**.
4. **LangChain** orchestrates AI workflows.
5. A **Large Language Model (LLM)** generates personalized mindfulness messages to guide users toward healthier digital habits.

---

# Impact

DigitalCalm SL helps users:

- Become **more aware of smartphone usage patterns**
- Reduce **digital fatigue and distraction**
- Build **healthier digital habits**
- Receive **data-driven mindfulness reminders**

By combining **behavioral analytics with AI-generated insights**, DigitalCalm SL promotes sustainable digital well-being.

# Tech Stack

- **React Native (Expo)** – Mobile application development
- **Firebase** – Backend infrastructure and serverless functions
- **Java** – Android system interaction and usage tracking
- **Neo4j** – Graph database for behavioral pattern detection
- **LangChain** – AI workflow orchestration
- **Large Language Models (LLMs)** – Personalized message generation
